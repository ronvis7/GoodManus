## 3. ReAct循环详解

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                          ReActAgent.execute_step()                              │
│                                                                                 │
│   输入: 用户消息 + 当前步骤描述                                                   │
│                                                                                 │
│       │                                                                         │
│       ▼                                                                         │
│   ┌─────────────────────────────────────────────────────────────────────────┐  │
│   │                     第1轮 LLM调用                                        │  │
│   │  ─────────────────────────────────────────────────────────────────────  │  │
│   │  系统提示词: 你是一个任务执行智能体                                      │  │
│   │              当前任务: 创建Python文件                                    │  │
│   │              可用工具: shell_execute, browser_navigate, ...             │  │
│   │                                                                         │  │
│   │  用户输入: 创建一个名为fib.py的文件                                      │  │
│   │                                                                         │  │
│   │  LLM输出: {                                                              │  │
│   │    "content": "",  // 空内容，表示需要调用工具                          │  │
│   │    "tool_calls": [{                                                      │  │
│   │      "id": "call_1",                                                     │  │
│   │      "function": {                                                       │  │
│   │        "name": "shell_execute",                                          │  │
│   │        "arguments": '{"command": "touch fib.py"}'                        │  │
│   │      }                                                                   │  │
│   │    }]                                                                    │  │
│   │  }                                                                       │  │
│   └─────────────────────────────────────────────────────────────────────────┘  │
│       │                                                                         │
│       ▼                                                                         │
│   ┌─────────────────────────────────────────────────────────────────────────┐  │
│   │                      工具调用                                            │  │
│   │  1. 解析工具调用参数                                                    │  │
│   │  2. 查找对应工具: ShellTool                                            │  │
│   │  3. 调用: shell_execute(command="touch fib.py")                        │  │
│   │  4. 执行: 向沙箱发送HTTP请求                                           │  │
│   │  5. 返回: ToolResult(success=True, message="文件已创建")               │  │
│   └─────────────────────────────────────────────────────────────────────────┘  │
│       │                                                                         │
│       ▼                                                                         │
│   ┌─────────────────────────────────────────────────────────────────────────┐  │
│   │                     第2轮 LLM调用（带上工具结果）                        │  │
│   │  ─────────────────────────────────────────────────────────────────────  │  │
│   │  消息历史:                                                               │  │
│   │    [user] 创建一个名为fib.py的文件                                      │  │
│   │    [assistant] tool_calls: shell_execute                                │  │
│   │    [tool] 文件已创建                                                    │  │
│   │                                                                         │  │
│   │  LLM输出: {                                                              │  │
│   │    "content": '{                                                         │  │
│   │      "success": true,                                                    │  │
│   │      "result": "已成功创建fib.py文件",                                 │  │
│   │      "attachments": ["/home/ubuntu/fib.py"]                            │  │
│   │    }',                                                                   │  │
│   │    "tool_calls": null  // 无工具调用，任务完成                          │  │
│   │  }                                                                       │  │
│   └─────────────────────────────────────────────────────────────────────────┘  │
│       │                                                                         │
│       ▼                                                                         │
│   ┌─────────────────────────────────────────────────────────────────────────┐  │
│   │                     解析结果                                             │  │
```
